This project aims to solve The Producer-Consumer Problem; and it consists of 3 python files that will run CONCURRENTLY (on 3 separate terminals) 

Steps to running this program:
1. Open the server.py file
1.1 Open terminal and run the command; *python server.py*

2. Open the producer_client.py file
2.1 Open a second terminal under this file and run the command; *python producer_client.py*

3. Open the consumer_client.py file
3.1 Open the third and final terminal under this file and run the command; *python consumer_client.py*
